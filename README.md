# Personal DevPortfolio & Blog Page

##  Project Description
A premium, dark-themed responsive portfolio and personal blog system engineered for modern software developers. This project serves as an academic coursework submission for **CSD 34203: Special Topics in Software Development** at **Universiti Sultan Zainal Abidin (UniSZA)**, Faculty of Informatics and Computing (FIK). It showcases a clean, interactive user interface designed to present academic milestones, technical proficiency progress metrics, and professional insights.

---

##  Features
- **Modern Responsive Layout:** Fully optimized fluid grid layout designed using modern CSS architecture (`Grid` and `Flexbox`) that responds seamlessly across Desktop, Tablet, and Mobile devices.
- **Dynamic Theme Toggle Engine:** Native JavaScript interface implementation allowing seamless transitions between a sleek **Dark Mode** (Premium Matrix Theme) and **Light Mode** with persistent client-side storage implementation (`localStorage`).
- **Interactive Component Systems:** Built-in dynamic skill-bar progress visualization, horizontal card matrices for academic roadmaps, and micro-interaction focus tags.
- **Micro-interactions:** Smooth transition engine animations covering all background color shifts (`0.25s ease`) and interactive link/button states (`0.15s ease`).

---

##  Technologies Used
- **HTML5:** Semantic markup scaffolding structure (`<header>`, `<main>`, `<aside>`, `<section>`, `<footer>`) to maximize accessibility and SEO architecture.
- **CSS3 Core:** Custom CSS variable trees (`:root` property design systems), responsive media queries (`@media`), and performance-optimized transition matrices.
- **Vanilla JavaScript (ES6+):** Light/Dark mode state management, client DOM rendering, and active browser event listeners.
